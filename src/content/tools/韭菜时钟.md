# 韭菜时钟

## 工具介绍

韭菜时钟是一款基于市场情绪和周期变化的智能分析工具，通过多种指标综合判断当前市场所处的阶段，帮助投资者把握最佳的买卖时机。

## 核心功能

- **市场情绪监测**：实时监测市场情绪指标，包括恐慌指数、贪婪指数等
- **周期阶段判断**：自动识别市场所处的周期阶段，如启动期、发酵期、高潮期、退潮期
- **买卖信号**：根据市场阶段提供明确的买卖信号
- **历史数据分析**：分析历史周期规律，提高判断准确性
- **个性化设置**：支持根据个人交易风格调整参数

## 使用方法

1. 打开工具后，系统会自动加载最新的市场数据
2. 查看当前市场所处的周期阶段和情绪指标
3. 根据工具提供的信号制定交易策略
4. 可设置提醒，在关键节点获得通知

## 适用人群

- 短线交易者
- 波段操作者
- 希望提高择时能力的投资者

## 技术特点

- 采用多因子模型分析市场情绪
- 结合技术分析和量能分析
- 实时数据更新，确保判断准确性
- 简洁直观的界面设计，易于使用
