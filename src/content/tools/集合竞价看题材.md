# 集合竞价看题材

## 工具介绍

集合竞价看题材功能如名字

## 核心功能

- **题材识别**：自动识别集合竞价中的热点题材和板块
- **强度排名**：对题材和板块进行强度排名，找出最强热点
- **历史对比**：与历史集合竞价数据进行对比，发现规律

## 使用方法

1. 早上开盘前启动集合竞价看题材工具
2. 实时查看集合竞价数据和分析结果
3. 关注热点题材和资金流向变化
4. 根据分析结果制定当日交易策略
